# ✨ Key Features

## 🦐 Managed Shrimp (Anomalocaris)
- **Lifecycle Management**: Start, stop, restart, and delete execution units via Discord or Web UI.
- **Persistent Workspace**: Every instance gets a dedicated host-mounted directory for file persistence.
- **Proactive Mailbox**: Anomalocaris instances communicate back to the host via a mailbox system for automatic status updates.
- **Docker Pruning**: Integrated tools to clean up "zombie" containers and reclaim system resources.

## 🎓 Skill Learning System
- **Auto-Capture**: Captures substantial code files (prioritizing logic over artifacts).
- **MCP Integration**: Learned skills become native Model Context Protocol tools immediately.
- **Metadata Rich**: Skills include auto-generated descriptions and parameter schemas.

## 🧠 Reasoning & Planning
- **Multi-step Reasoning**: Chains complex operations across search, files, and code.
- **Smart Variable Passing**: Outputs from one tool serve as inputs for the next.
- **Provider Agnostic**: Switch seamlessly between **Google Gemini 2.0** and **Anthropic Claude 3.5**.

## 📚 Hybrid RAG System
- **Multi-Source Knowledge**: Combines local ChromaDB with Pinecone Cloud for infinite memory.
- **Identity Linking**: Unified memory across Discord, LINE, and Web.

## 🕒 Robust Daily Scheduler
- **Multi-Platform Delivery**: Unified result rendering and dynamic delivery across both Discord and LINE.
- **Identity-Aware**: Accurately tracks canonical user IDs and principal maps to ensure private data (like calendars and emails) is securely routed.
